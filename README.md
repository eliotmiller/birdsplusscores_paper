# BirdsPlus Species Scores

This repository contains the data and code necessary to duplicate the **BirdsPlus species scores** that accompany our in-review paper.

**Preprint:** <https://www.biorxiv.org/content/10.1101/2025.07.15.664934v1>

------------------------------------------------------------------------

## Repository Contents

-   **`data/`** — Contains all input datasets used by the scripts.\
-   **`R/`** — Contains supporting R code/functions. Specifically the\
-   **`v1_4_speciesScores.R`** — Main R script used to generate the species scores.\
-   **Other scripts** — And also other scripts to generate intermediate data products (including migration distance). Included here for completeness.\

------------------------------------------------------------------------

## Usage

1.  Clone the repository:

``` bash
git clone https://github.com/YOURUSERNAME/YOURREPO.git
```

2.  Open v1_4_speciesScores.R in R and run the script to generate the species scores.

3.  Optional: Run the other scripts to reproduce intermediate data products.
